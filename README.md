📘 Finance Tracker — Short Project Summary

A full-stack personal finance management application built using Next.js, Node.js, Express, and MongoDB.
Users can manage income and expenses, track balance, update their profile, and reset passwords with email verification.

🚀 Features

User signup & login (JWT authentication)

OTP-based email verification

Forgot/reset password

Add, edit, delete transactions

Sort & filter transactions

View total income, expense & balance

User profile with change-password option

Fully responsive UI (TailwindCSS)

🛠 Tech Stack

Frontend: Next.js, React, TailwindCSS, Axios
Backend: Node.js, Express.js, MongoDB
Auth: JWT, BCrypt
Deployment: Vercel (Frontend), Render (Backend)

📂 Project Structure
backend/   → API, routes, controllers, models
frontend/  → Next.js app, pages, components, UI, API integration

▶️ How It Works

User creates account → verifies via OTP

User logs in → receives secure access token

Dashboard loads personal transactions

User can create, filter, edit, or delete entries

Profile section allows password update & logout
